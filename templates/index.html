Overview

Purpose
The purpose of this project was to perform an exploratory analysis of Airbnb's rental landscape in Asheville, North Carolina.This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
Data Source
The dataset for the analysis was sourced from Inside Airbnb and consists of:
- Listings: Details of all Airbnb listings in Asheville with 106 attributes for each listing.
- Reviews: Provides detailed reviews given by the guests with attributes - Date, listing_id, reviewer_id & comment.
- Calendar: Provides details of available booking times by listing_id, date, availability & price.
